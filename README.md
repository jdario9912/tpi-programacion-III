### Ejecutar el proyecto

**Linux o Mac** 

Dar permisos de ejecución al script

```sh
chmod +x run.sh
```

Ejecutar el script

```sh
./run.sh
```

**Windows**

Ejecutar el script en CMD

```cmd
run.bat
```

O hacer doble clic en `run.bat`.

Ejecutar el script en PowerShell

```powershell
.\run.ps1
```
